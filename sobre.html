Sobre o projeto

Este siste foi desenvolvolvido como parte do projeto da disciplina EXA844 programação para redes do curso de Engenharia da Computação da Universidade Estadual de Feira de Santana( UEFS )

O projeto consistiu em três etapas:

  • Crawler: realizar a busca e raspagem de um fonte na internet para montar uma base de dados estruturada
  • API REST: Construção e hospedagem de um API rest para consulta dos dados
  • Interface web: Construção de uma interface web para as consultas da API
